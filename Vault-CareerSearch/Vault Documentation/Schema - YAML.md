App notes need to be titled as follows:
`[Company Name] - [Role Title] (App)`

## (App) Notes YAML

```
---
type: App
status: Open
created: 
company: 
role: 
listing: 
resume: 
cover_letter: 
recruiter_call: 
first_interview: 
second_interview: 
offer_date: 
close_date: 
---
```

## (Listing) Note YAML
```
---
Type: "Listing"
"date created":
Company:
Role:
URL:
App:
---
```
